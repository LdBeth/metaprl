Using a custom editor

Editing files in the browser gets old fast. However, browser security prevents us from invoking an external editor to edit these files. However, it is still possible! Here is how to do it.

The strategy is to get your Browser to handle MetaPRL files with Content-Type: application/x-metaprl.

  • First, you need to download the medit application. Download it to a standard place.
    • Linux
    • Microsoft Windows
  • Next, choose the External Editor option in the View menu.
  • When you try to edit a file, your browser should now prompt you for what to do. Choose the medit application you just saved.
  • By default, medit will invoke vi on Unix, and notepad on Windows. To change this, edit the file ~/.metaprl/editor.
